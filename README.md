# Why this project?

This is a little example how to set up cucumber within a Spring Boot Reactive Web environment.

# Build and run test

## Build

```shell script
$ ./mvnw clean compile
```

## Run tests

```shell script
$ ./mvnw clean test
```

## Build package

```shell script
$ ./mvnw clean package
```

## Run

```shell script
$ java -jar target/webflux-example.jar
```
